# 🛒 CodeAlpha E-commerce Store

A full-stack MERN E-commerce Store developed as part of the **CodeAlpha Internship Program**.

## 📌 Features

* 👤 User Registration & Login
* 🛍️ Product Listing
* 📄 Product Details Page
* 🛒 Shopping Cart
* ➕➖ Increase / Decrease Product Quantity
* 🔍 Product Search
* 📂 Category Filter
* 💳 Checkout Page
* 📦 Order Processing
* 🔔 Toast Notifications
* 📱 Responsive UI using Bootstrap

## 🛠️ Tech Stack

### Frontend

* React.js
* Bootstrap
* React Router
* Axios
* React Toastify

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

## 📂 Project Structure

```text
CodeAlpha_EcommerceStore/
│
├── frontend/
│
└── backend/
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Komal0124-byt/CodeAlpha_EcommerceStore.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
